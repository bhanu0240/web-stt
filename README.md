# web-stt

# npm i

# install ffmpeg on your computer (https://www.gyan.dev/ffmpeg/builds/)

# create a Hugging face account and generate access_token with below permissions

# - Read access to contents of all repos under your personal namespace

# - Read access to contents of all public gated repos you can access

# - Make calls to Inference Providers

# - Access webhooks data

# - Start and manage Jobs on your behalf

# create a .env file at the root of the project and set HF_TOKEN of your account

# npm run start will run your both frontend and backend locally
